Pasos para Ejecutar el Archivo .JAR del proyecto
1) Abrir la carpeta donde se encuentra el archivo .JAR
2) Escribir cmd en la parte superior del Explorador de Archivos, donde se muestra la ubicación de la carpeta en el equipo
3) Escribir en el cmd: java -jar Calculadora_T4.jar
   *Calculadora_T4.jar = nombre del archivo
4) Dar ENTER
5) Se procederá a ejecutar el proyecto la terminal del dispositivo
